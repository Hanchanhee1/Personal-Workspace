# 라이프 대시보드 - 프론트엔드

React + TypeScript + Vite로 제작된 개인용 라이프 대시보드입니다.

## ⚠️ 개발 규칙

- 소스 코드의 모든 주석은 **한국어**로 작성해야 합니다.
- 새로운 라이브러리 추가 시 이 문서의 기술 스택 섹션을 업데이트하세요.

## 기능

- **맛집 지도**: 내가 저장한 맛집 위치를 지도에 표시합니다.
- **일기장**: 하루의 기록을 남기고 확인할 수 있습니다.
- **캘린더**: 월별 일정을 관리합니다.
- **할 일 목록(Todo)**: 해야 할 일을 체크리스트로 관리합니다.
- **날씨**: 현재 날씨 정보를 보여줍니다.
- **글래스모피즘 디자인**: 현대적이고 세련된 UI를 제공합니다.

## 🚀 개발 환경 설정 및 실행 방법

이 프로젝트를 로컬에서 실행하기 위해 다음 명령어를 순서대로 입력하세요.

### 1. 프로젝트 폴더로 이동 (이미 frontend 폴더 안이라면 건너뛰기)
```bash
cd frontend
```

### 2. 패키지 설치
`package.json`에 정의된 라이브러리들을 설치합니다.
```bash
npm install
```

### 3. 개발 서버 실행
로컬 개발 서버를 킵니다.
```bash
npm run dev
```

### 4. 빌드 (배포용)
프로덕션용 정적 파일을 생성합니다.
```bash
npm run build
```
